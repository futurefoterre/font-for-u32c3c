安装指南
选择你的平台
根据使用场景选择对应安装方式。macOS 支持完整使用，Windows 支持浏览器与特定应用，Android 已Root设备可完整支持，iOS 与未Root Android 目前仅限本页书写器。
macOS
桌面端字体
安装后此字符可在所有 macOS 原生应用中自动显示，包括备忘录、Pages、Sublime Text 等，无需手动设置字体。
⬇ 下载字体
安装字体：双击 .otf 文件，点击「安装字体」即可。安装后所有应用自动支持，无需重启。

输入方式：前往「系统设置 → 键盘 → 文字替换」→「+」。「替换为」栏粘贴 𲰼（点击书写器中的「复制文字」），「快捷键」填写拼音，例如 ta。之后在任意应用中输入快捷键并按空格即可插入此字符。
浏览器 · Chrome / Firefox / Arc / Edge
Stylus 一键样式
安装后可在浏览器中所有网页正确显示此字符。适用于 macOS 与 Windows。
第一步 · 先安装扩展
① 安装 Stylus 扩展
第二步 · 再安装样式
② 一键安装 Stylus 样式
⚠️ 注意：Stylus 样式会将所有网页字体替换为系统字体栈，可能影响部分网站的排版外观。建议仅在需要显示此字符时启用，在 Stylus 扩展图标中可按网站单独开关。
Windows
Windows 使用说明
Windows 字体回退机制与 macOS 不同，安装字体后不会自动在所有应用中显示此字符，需根据场景分别配置。
⬇ 下载字体
安装字体
① 下载字体文件
② 右键点击 .ttf 文件 → 选择「为所有用户安装」
③ 重启电脑后生效
输入方式
Windows 无系统级文字替换功能，目前无法像 macOS 一样通过拼音自然输入此字符。建议从本页书写器复制后粘贴使用。
浏览器（推荐）· Stylus
安装 Stylus 扩展及本页样式后，所有网页均可正确显示此字符。详见上方浏览器卡片的安装步骤。
Word / LibreOffice 等
粘贴字符后选中显示为方块的部分，手动将字体改为「Noto Sans CJK SC Patched」即可正确显示。
⚠️ 记事本、微信等无字体选择功能的原生应用暂时无法显示此字符，待 Microsoft 更新系统字体支持 Unicode 17.0 后将自动改善。
Android
Android 安装说明
根据设备情况选择安装方式。已Root设备可实现所有应用完整支持；未Root设备可尝试 /data/fonts/ 方法或使用本页书写器。
✓ 已Root设备 · Magisk 模块（推荐） ⬇ 下载 Magisk 模块 安装后所有应用（包括微信、小红书等）均可正确显示，不影响微信/QQ正常使用。

① 将 .zip 文件复制到手机
② 打开 Magisk → 底部「模块」→ 右上角「从本地安装」
③ 选择下载的 .zip 文件
④ 等待安装完成 → 点击「重启」
⑤ 重启后所有应用即可显示字形 ✓

如需了解如何 root 设备，请参考 XDA Developers 上针对具体机型的教程。
⚠️ root 有变砖风险，请谨慎操作。
△ 未Root · /data/fonts/ 方法（待验证） ⬇ 下载字体文件 部分机型（如小米）可能支持此方法，但尚未在无Root环境下完整验证。效果因机型而异。

① 用文件管理器进入 /data/fonts/
② 若路径存在，将字体文件粘贴进去
③ 重启手机
④ 测试备忘录与浏览器是否显示字形

⚠️ 此路径在部分机型不存在。即使成功，小红书、微信等自带字体的应用仍不支持。欢迎在 GitHub 反馈测试结果。
iOS · iPhone / iPad
iPhone 字体配置文件
提前安装，为未来做准备。当 iOS 系统更新支持 Unicode 17.0 第三平面字符时，此字符将在所有原生应用中自动正确显示。需要 iOS 14 或更高版本。
⬇ 安装配置文件
方法一 · 直接在 iPhone 上点击下载按钮：
① 在 iPhone Safari 中打开本页面
② 点击「安装配置文件」按钮，弹出提示后点击「允许」
③ 打开「设置」→ 顶部「已下载描述文件」→ 点击「安装」→ 输入锁屏密码 →「完成」

方法二 · 从 Mac AirDrop：
① 在 Mac 上右键 .mobileconfig 文件 → 「共享 → AirDrop」→ 选择 iPhone
② 前往「设置」顶部「已下载描述文件」完成安装

⚠️ 受限于 iOS 系统对 Unicode 17.0 第三平面字符的支持尚未完善，此配置文件目前在原生应用中实际效果有限。建议提前安装以便系统更新后自动生效。
输入方式
如何在各平台输入此字符
由于 U+32C3C 位于 Unicode 17.0 第三表意文字平面，各平台的输入与显示支持程度不同。
macOS · 文字替换快捷输入（推荐）
前往「系统设置 → 键盘 → 文字替换」→「+」。「替换为」栏粘贴 𲰼（点击书写器中的「复制文字」），「快捷键」栏填写拼音，例如 ta。之后在任意 macOS 应用中输入该快捷键并按空格，即可插入此字符。
浏览器（需安装 Stylus）
安装 Stylus 样式后，所有网页均可正确渲染此字符。适用于 macOS 与 Windows 浏览器。配合文字替换快捷键（macOS），可在网页输入框中直接输入并显示。
iOS · 仅限本页书写器
iOS 系统 Core Text 目前尚不支持 Unicode 17.0 第三平面字符，在原生应用（如微信、备忘录、短信）中无法输入或正常显示此字符。待 iOS 后续系统版本更新后将自动支持。
Android · 已Root设备完整支持 / 未Root仅限本页书写器
已root并安装 Magisk 模块的用户，所有应用均可显示此字符，可配合输入法自定义词库输入。未root用户请使用本页书写器输入与预览，待 Google 更新系统字体后将自动改善。
